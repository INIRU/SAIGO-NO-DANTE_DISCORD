## 자동 매핑 코드

### 키워드 매칭 규칙

텍스트에서 키워드를 자동 인식할 때 아래 규칙을 따른다:

1. **긴 키워드 우선 매칭** — 길이 내림차순 정렬. "진동 폭발"이 "진동"보다 먼저 매칭.

2. **짧은 키워드(2글자 이하) 경계 체크** — `못`, `관` 같은 짧은 키워드:
   - **앞글자가 한글이면 매칭 안 함**
   - **뒷글자가 한글이면 매칭 안 함** — 단, 뒷글자가 **조사**이면 매칭 허용

3. **조사 허용 목록**: 이, 가, 을, 를, 의, 에, 은, 는, 과, 와, 도, 로, 만, 야, 라, 며, 든, 인, 및, 나, 란

4. **3글자 이상**은 경계 체크 없이 매칭.

5. **숫자 후행 매칭** — "출혈 3" → 키워드 + 수치

function isHangul(ch) {
  if (!ch) return false;
  const code = ch.charCodeAt(0);
  return (code >= 0xAC00 && code <= 0xD7A3) || (code >= 0x3131 && code <= 0x318E);
}

const PARTICLES = new Set(['이','가','을','를','의','에','은','는','과','와','도','로','만','야','라','며','든','인','및','나','란']);

function keywordToEmoji(keyword) { return KEYWORD_EMOJI_MAP[keyword] ?? keyword; }

function replaceKeywordsWithEmoji(text) {
  const allKeywords = Object.keys(KEYWORD_EMOJI_MAP).sort((a, b) => b.length - a.length);
  let result = '', prevChar = '';
  while (text.length > 0) {
    let found = false;
    for (const kw of allKeywords) {
      if (!text.startsWith(kw)) continue;
      if (kw.length <= 2) {
        if (isHangul(prevChar)) continue;
        const nextCh = text[kw.length];
        if (isHangul(nextCh) && !PARTICLES.has(nextCh)) continue;
      }
      result += KEYWORD_EMOJI_MAP[kw]; prevChar = kw[kw.length - 1]; text = text.slice(kw.length); found = true; break;
    }
    if (!found) { result += text[0]; prevChar = text[0]; text = text.slice(1); }
  }
  return result;
}
